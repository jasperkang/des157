Studio 5: The Library

February 21 2017


For this studio, I used AngularJS to create a list with a user selected filter. Compared to another library such as Jquery, AngularJS is much more simple and powerful. It is based entirely on HTML and Javascript, so there is no need to learn a new language. It is advertised as "What HTML should have been if it was created for web applications with rich interaction", and this is true. It is useful for creating complex web apps with user interactions.
