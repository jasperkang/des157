Studio 4: Time is on my side

February 14 2017


For this studio, I used three different countdown timers. The user can choose the countdown for Spring Break, Christmas, or New Years with the buttons along the bottom. The countdown is organized into days, hours, minutes, and seconds, which the user can see clearly with the four separated elements. When the user clicks each button, the colors of the interface change, and the background image also changes. It is a simple and effective interface that most users should understand immediately.
